# 📖 Project Overview

Businesses generate large amounts of customer shopping data every day. However, raw data alone cannot support strategic decision-making.

This project analyzes customer purchasing behavior using **Python, SQL Server, and Power BI** to uncover trends in customer demographics, spending patterns, product categories, subscription behavior, and sales performance. The insights help businesses understand customer preferences and make informed business decisions.

---

# 🎯 Business Problem

This project answers important business questions such as:

- Which product category generates the highest revenue?
- Which age group contributes the most sales?
- How do subscription customers differ from non-subscribers?
- Which shipping methods are most preferred?
- What purchasing patterns can improve business performance?

# 💡 Key Insights

- 👕 Clothing generated the highest revenue.
- 🧑 Young Adults contributed the highest purchase amount.
- 📦 Accessories ranked second in overall sales.
- 💳 Most customers were non-subscribers.
- ⭐ Average customer rating remained around **3.75**.
- 💲 Average purchase amount was approximately **$59.76**.

# ▶️ How to Run

### Clone the Repository

```bash
git clone https://github.com/yourusername/Customer-Behavior-Analysis.git
```

### Install Dependencies

```bash
pip install pandas numpy matplotlib seaborn
```

### Run the Project

1. Open the Jupyter Notebook.
2. Execute the Python notebooks.
3. Run the SQL queries in SQL Server Management Studio (SSMS).
4. Open the Power BI dashboard (`.pbix`) in Power BI Desktop.
