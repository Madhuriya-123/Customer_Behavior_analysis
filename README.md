# рџ“– Project Overview

Businesses generate large amounts of customer shopping data every day. However, raw data alone cannot support strategic decision-making.

This project analyzes customer purchasing behavior using **Python, SQL Server, and Power BI** to uncover trends in customer demographics, spending patterns, product categories, subscription behavior, and sales performance. The insights help businesses understand customer preferences and make informed business decisions.
